# Словарь утилитарных классов uStatic CSS

Полный список всех доступных утилитарных классов библиотеки uStatic CSS.

---

## 📐 Align — Выравнивание элементов

| Класс | Описание |
|-------|----------|
| `.align-baseline` | Выравнивание по базовой линии текста |
| `.align-top` | Выравнивание по верху родителя |
| `.align-middle` | Выравнивание по середине родителя |
| `.align-bottom` | Выравнивание по низу родителя |

**Применение:** Для выравнивания inline и inline-block элементов (иконки в тексте, изображения в строке).

## 🎬 Animations — Анимации

| Класс | Описание |
|-------|----------|
| `.blink` | Мигание (opacity 50%, бесконечное повторение, 1s) |
| `.animation:spin-1` | Вращение, длительность 0.25с |
| `.animation:spin-2` | Вращение, длительность 0.5с |
| `.animation:spin-3` | Вращение, длительность 0.75с |
| `.animation:spin-4` | Вращение, длительность 1с |
| `.animation:spin-5` | Вращение, длительность 1.25с |
| `.animation:spin-6` | Вращение, длительность 1.5с |
| `.animation:spin-7` | Вращение, длительность 1.75с |
| `.animation:spin-8` | Вращение, длительность 2с |

## 🎨 Bg — Фон

| Класс | Описание |
|-------|----------|
| `.bg-{color}` | Цвет фона из палитры (например, `bg-primary`, `bg-gray-500`) |
| `.bg-{color}-{type}` | Цвет фона с указанием яркости (например, `bg-red-500`, `bg-blue-300`) |
| `.bg-none` | Прозрачный фон |
| `.bg-opacity-0` | Прозрачность фона 0% |
| `.bg-opacity-25` | Прозрачность фона 25% |
| `.bg-opacity-50` | Прозрачность фона 50% |
| `.bg-opacity-75` | Прозрачность фона 75% |
| `.bg-opacity-100` | Прозрачность фона 100% |
| `.bg-filter-blur-{1-10}` | Размытие фона (backdrop-filter: blur 1-10px) |
| `.hover:bg-{color}` | Фон при наведении |
| `.hover:bg-{color}-{type}` | Фон при наведении с указанием яркости |

**Применение:** Фоны кнопок, карточек, секций, оверлеев.

## 🖼️ Border — Границы и скругления

|-------|----------|

| `.border-0` | Граница толщиной 0px |
| `.border-1` | Граница толщиной 4px |

| `.border-r`, `.border-r-{1-5}` | Правая граница |
| `.border-b`, `.border-b-{1-5}` | Нижняя граница |
| `.border-l`, `.border-l-{1-5}` | Левая граница |
| `.border-{color}` | Цвет границы (например, `border-primary`, `border-gray-300`) |
| `.border-{color}-{type}` | Цвет границы с яркостью (например, `border-red-500`, `border-blue-300`) |
| `.border-transparent` | Прозрачная граница |
| `.border-dashed` | Пунктирная граница |
| `.divider-x` | Горизонтальный разделитель (высота 1px) |
| `.divider-y` | Вертикальный разделитель (ширина 1px) |

| `.rounded-sm` | Скругление 2px |
| `.rounded-base` | Скругление 4px (базовое) |
| `.rounded-md` | Скругление 6px |
| `.rounded-lg` | Скругление 8px |
| `.rounded-xl` | Скругление 12px |

| `.rounded-full` | Полное скругление (9999px) |

| `.rounded-tr-*`, `.rounded-tl-*`, `.rounded-br-*`, `.rounded-bl-*` | Скругление одного угла |

**Примечание:** Класс `.rounded` без суффикса не существует — используйте `.rounded-base` для скругления по умолчанию.

| `.text-none` | Прозрачный текст |
| `.text-decoration-none` | Без украшений |
| `.text-ellipsis` | Обрезка с многоточием |
| `.hover:text-{color}` | Цвет при наведении |
| `.hover:text-{color}-{type}` | Цвет при наведении с указанием яркости |
| `.line-height-{size}` | Высота строки (соответствует размерам текста) |
| `.italic` | Курсив |
| `.underline` | Подчеркивание |
| `.nowrap` | Без переноса строк |
| `.pre-wrap` | Сохранение пробелов и переносов |
| `.break-word` | Перенос длинных слов |
| `.list-style-none` | Без маркеров |
| `.truncate` | Обрезка текста (многоточие, max-width: 100%) |
| `.list-reset` | Сброс стилей списка |

**Применение:** Заголовки, параграфы, ссылки, списки, многострочный текст.

## Примеры использования

### Комбинирование классов

```html
<!-- Кнопка -->
<button class="flex items-center justify-center gap-2 p-4 bg-blue-500 text-white rounded-base hover:bg-blue-600 cursor-pointer">
  Нажми меня
</button>

<!-- Карточка -->
<div class="block p-6 bg-white border border-gray-200 rounded-base hover:opacity-75">
  <h3 class="text-lg font-bold text-gray-800">Заголовок</h3>
  <p class="text-gray-600 mt-2">Описание карточки</p>
</div>

<!-- Бейдж -->
<span class="inline-block px-2 py-1 text-xs font-semibold text-red-600 bg-red-100 rounded-base">
  New
</span>
```

### Псевдоклассы

- `.hover:bg-blue-500` — фон при наведении
- `.hover:text-white` — цвет текста при наведении
- `.active:pulse` — пульсация при нажатии

### Цвета и яркость

Цвета указываются с яркостью через дефис:
- `bg-red-500`, `bg-blue-300`, `bg-green-700`
- `text-gray-600`, `text-primary-400`
- `border-purple-500`, `border-orange-200`

### Размеры отступов

Значения следуют шкале spacing: `0, 1, 2, 4, 8, 12, 16, 20, 24, 32, 40, 48, 64px` и т.д.
